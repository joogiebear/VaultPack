# VaultPack

Welcome to the official VaultPack documentation!

## Overview

VaultPack is a powerful multi-backpack storage system for Minecraft servers that provides players with expandable inventory space through backpacks and ender chest pages. With a robust economy integration, crafting system, and extensive customization options, VaultPack is the ultimate storage solution for survival servers.

## Key Features

### Multiple Backpack Slots
- Up to 18 backpack slots per player
- Each slot can hold a different backpack tier
- Permission-based or economy-based unlocking

### Backpack Tiers
- **Small Backpack** - 9 slots (1 row)
- **Medium Backpack** - 18 slots (2 rows)
- **Large Backpack** - 27 slots (3 rows)
- **Greater Backpack** - 36 slots (4 rows)
- **Jumbo Backpack** - 45 slots (5 rows)

### Ender Chest Pages
- Up to 9 ender chest pages
- Each page contains 27 slots
- Expandable through economy or permissions

### Unified Storage Menu
- Access all backpacks and ender chest pages in one GUI
- Intuitive navigation system
- Customizable appearance

### Economy Integration
- Vault support for unlocking slots and pages
- Configurable upgrade costs
- Permission-based bypass options

### Crafting System
- Craft backpacks with custom recipes
- Upgrade backpacks to higher tiers
- EcoItems integration for custom materials

### Death Protection
- Keep backpack items on death (configurable)
- Keep ender chest items on death
- Respects keepInventory gamerule

### API for Developers
- Events for backpack actions
- Programmatic access to backpack data
- Integration-friendly design

## Soft Dependencies

VaultPack integrates seamlessly with these plugins:
- **Vault** - For economy features
- **PlaceholderAPI** - For placeholders in GUIs and chat
- **EcoItems** - For custom item support in recipes
